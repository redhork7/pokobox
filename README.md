# pokobox
redhork7의 redwish 포트폴리오 웹사이트
<br/>
https://pokobox.redhork7.com

## 소개
redwish에서 앞으로 제공하게 될 다양한 서비스와 라이브러리를 소개할 포트폴리오 웹사이트입니다.

## 기술 개요
- React SPA 웹사이트입니다.
- rolldown으로 빌드(번들링)합니다.
- tailwindcss, radix, shadcn으로 UI/UX 컴포넌트를 구현합니다.
- 도메인은 AWS route53으로 라우팅됩니다.
- 호스팅은 클라우드플레어로 제공됩니다.

## 명령어
```shell
# 개발환경 실행
$ bun run dev

# 빌드
$ bun run build

# 배포
$ bun run deploy
```
